Selectores

Tipo de selectores que entran en un examen

Clase (class="")

El atributo global class se trata de una lista de las clases de los elementos separada por espacios. Las clases permiten a CSS y Javascript seleccionar y acceder a elementos específicos a través de los selectores de clase o funciones.


Ejemplo:hola


Identificador (id="")

El atributo global id define un identificador único (ID) el cual no debe repetirse en todo el documento. Su propósito es identificar el elemento al vincularlo usando un identificador de fragmento, en scripts u hojas de estilo con CSS.


Ejemplo: new example


Todos los elementos (*)

Este atributo personaliza todos los elementos de la página evitandonos tener que ir uno a uno cambiando su estilo.
